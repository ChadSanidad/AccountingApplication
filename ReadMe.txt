To run
:
Preconditions: The WKIS database must be loaded.
The Assignment 2 sql must be loaded.


Run the Integration Driver, 

Step 1. Log into the database, if a user is already running the program,
The program will notify that its already in use. 

Step 2. Choose a bin file to load into the database 
Then create a txt file to convert the bin file to.

Step 3. Push perform month end to zero out the balances on the database.

Step 4. Choose a folder to export a report to. Then choose the alias of the file, lastly
name the file of the report Ex( report.txt).
